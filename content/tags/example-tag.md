# @example

## Concept

The `@example` tag provides a code snippet demonstrating how to use the documented function, class, or constant.

## Implementation

```javascript
/**
 * @example
 * const user = greet("Alice");
 */
function greet(name) {
  return `Hello, ${name}`
}
```

## Checklist

    [ ] Is the code block valid and syntactically correct?

    [ ] Does the example illustrate the primary use case?

    [ ] Is the example concise (3–5 lines max)?
